First, run the development server:

```bash
npm install
# or
yarn
```

```bash
npm run dev
# or
yarn dev
```

Open [http://127.0.0.1:3000](http://127.0.0.1:3000) with your browser to see the result.

Running the code on ios simulator
```bash
yarn ios
```

Then, start the app via xcode
